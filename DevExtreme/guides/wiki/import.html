Самописная тулза для импорта страничек из гитхаба.

В приложенном архиве (/wikiimporter.zip) содержится файл проекта на C# и сбилженное приложение (в директории .\bin\debug)

Для запуска просто запустить экзешник:

  • WikiImporter\WikiImporter\bin\Debug\net6.0\WikiImporter.exe
     

Технически работает через запросы graphql к вики.

где:
